## Что такое EIA Storage Report?

**EIA Storage Report** — еженедельный отчёт о изменении запасов природного газа в подземных хранилищах США.

**Публикация:**  
- **День:** Каждый четверг (кроме праздников)
- **Время:** 10:30 AM EST = **18:30 МСК** (зимнее время) / **17:30 МСК** (летнее время)
- **Формат:** Изменение в billion cubic feet (bcf)

**Зачем важно:**  
Запасы газа — ключевой индикатор баланса спроса/предложения. Неожиданные изменения → резкая волатильность цены NG (±5-10% за минуты).

---

## Типы изменений и интерпретация

| Сезон | Нормальное изменение | Интерпретация | Влияние на цену NG |
|-------|----------------------|---------------|---------------------|
| **Зима (ноя-мар)** | **-100 до -200 bcf** | Withdrawal (вывод газа из хранилищ) для отопления | Нейтрально / +Bullish если > -200 |
| **Весна (апр-май)** | **+50 до +100 bcf** | Injection (закачка) начинается | Нейтрально |
| **Лето (июн-авг)** | **+70 до +120 bcf** | Активная закачка | Нейтрально / -Bearish если > +150 |
| **Осень (сен-окт)** | **+30 до +70 bcf** | Замедление закачки, подготовка к зиме | Нейтрально / +Bullish если < +30 |

## Формула расчёта "Сюрприза"

Рынок реагирует **НЕ на абсолютное значение**, а на **отклонение от прогноза**.

Сюрприз (bcf) = Фактическое изменение - Консенсус-прогноз

Где взять прогноз:

Bloomberg Survey (платный)

Trading Economics: https://tradingeconomics.com/united-states/natural-gas-stocks-change

### Пример:

**Дата:** 23 января 2026  
**Прогноз:** -140 bcf (withdrawal)  
**Факт:** -210 bcf  
**Сюрприз:** -210 - (-140) = **-70 bcf** (сильнее вывод, чем ожидали)  

**Интерпретация:**  
Газа вытащили БОЛЬШЕ, чем ожидалось → спрос выше прогноза → **Bullish** для цены NG

**Ожидаемая реакция:**  
```python
Движение цены (%) ≈ Сюрприз / 30
Движение = -70 / 30 = -2.3% → инвертируем знак для withdrawal = +2.3% рост

Trading Strategy вокруг EIA (для skill)
❌ ЧТО НЕ ДЕЛАТЬ:
НЕ входить в сделку за 2 часа до EIA (16:30-18:30 МСК)
→ Цена часто "замирает", потом резкий скачок (gap)

НЕ держать позиции через EIA без stop-loss
→ Риск проскальзывания -10% за минуту

НЕ торговать в первые 5 минут после EIA
→ Алгоритмы создают ложные пики (head fakes)

✅ ЧТО ДЕЛАТЬ:
Стратегия 1: "Post-EIA Stabilization" (Conservative)
text
Таймлайн:
18:30 МСК → EIA выходит
18:35 → Первая волна волатильности (игнорируем)
19:00 → Цена стабилизируется
19:15 → ВХОД в направлении сюрприза (если подтверждено)

Условия входа:
- Сюрприз > ±50 bcf
- Цена не развернулась после первой реакции (тренд продолжается)
- Confidence ≥ 65%

TP1: через 2-4 часа (+2-3% от входа)
SL: обратная сторона диапазона 18:30-19:00

def analyze_eia_impact(eia_report, current_price, direction):
    """
    eia_report: {'report_date': '2026-01-30', 'net_change_bcf': -210, 
                 'forecast_bcf': -140, 'total_storage_bcf': 2280}
    direction: 'long' или 'short' (текущее направление анализа)
    """
    
    # 1. Рассчитать сюрприз
    surprise_bcf = eia_report['net_change_bcf'] - eia_report['forecast_bcf']
    
    # 2. Определить силу сюрприза
    if abs(surprise_bcf) < 20:
        eia_impact = "neutral"
        confidence_adjustment = 0
    elif surprise_bcf < -50:  # Сильный withdrawal (bullish)
        eia_impact = "bullish" if surprise_bcf < -80 else "bullish_surprise"
        confidence_adjustment = +10 if direction == "long" else -10
    elif surprise_bcf > 50:  # Сильная injection (bearish)
        eia_impact = "bearish" if surprise_bcf > 80 else "bearish_surprise"
        confidence_adjustment = +10 if direction == "short" else -10
    else:
        eia_impact = "neutral"
        confidence_adjustment = 0
    
    # 3. Проверить свежесть отчёта
    days_old = (datetime.now() - eia_report['report_date']).days
    if days_old > 7:
        eia_impact = "stale"  # Устаревший отчёт
        confidence_adjustment = 0
    
    # 4. Добавить предупреждение, если EIA скоро
    warning = None
    if is_thursday_before_1830_msk():
        warning = "EIA report due today 18:30 MSK - expect 10-15% intraday volatility"
    
    return {
        'eia_impact': eia_impact,
        'surprise_bcf': surprise_bcf,
        'confidence_adjustment': confidence_adjustment,
        'warning': warning
    }
